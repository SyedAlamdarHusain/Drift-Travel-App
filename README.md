# Installing Libraries:

## To install the required libraries enter the commands:

```bash
cd Drift-Travel-App
```

```bash
py -3 -m venv venv
```

```bash
venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```

# Runing the program

## Enter the command:

```bash
  python run.py
```

or 

```bash
flask run
```
